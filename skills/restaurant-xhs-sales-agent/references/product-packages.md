## Package Recommendation Logic

- If the client asks “先试试” but has budget: recommend 12,000, because it is the balanced package and better reflects content quality.
- If the client has very limited budget: 9,500 can be used as a minimum test, but warn that lower budget means lighter resource quality.
- If the client wants stronger exposure, official flow, or faster momentum: recommend 16,650.
- If the client only asks for individual账号/笔记/攻略 distribution, quote single item references, but first confirm the goal and minimum effective quantity.
